# Character Creator

Open `character-sheet.html` to use the character creator.

- `feats.json` is the editable feat database.
- `mantles.json` is the editable mantle database.
- `character-saves/` is for saved character `.json` files.
- `pictures/` is for keeping source character images.

Saved character files include the selected character image inside the JSON, so loading a character save does not require the original picture file.
